# Landing Page

This repository contains a simple responsive landing page starter for the CMYK project.

Files added:
- `index.html` — main landing page
- `styles.css` — styles
- `script.js` — minimal JS for menu and smooth scroll

To use locally:
1. Clone the repo
2. Open `index.html` in your browser or serve with a local dev server

Feel free to modify colors, content, and assets to fit your brand.
